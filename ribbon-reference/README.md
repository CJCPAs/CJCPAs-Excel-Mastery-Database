## Quick Access Toolbar

The Quick Access Toolbar (QAT) appears above/below the ribbon. Default buttons:

| Button | Description | Shortcut |
|--------|-------------|----------|
| **Save** | Save workbook | Ctrl+S |
| **Undo** | Undo last action | Ctrl+Z |
| **Redo** | Redo undone action | Ctrl+Y |

### Customize QAT
Right-click QAT ‚Üí Customize Quick Access Toolbar
- Add any ribbon command
- Import/export customizations
- Show below/above ribbon
