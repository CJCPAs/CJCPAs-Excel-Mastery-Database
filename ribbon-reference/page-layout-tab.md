# Page Layout Tab - Complete Reference

> **Control printing, page setup, themes, and object arrangement**

---

## Tab Overview

The Page Layout tab controls how your worksheet looks when printed and on screen:

| Group | Purpose |
|-------|---------|
| [Themes](#themes-group) | Document-wide formatting |
| [Page Setup](#page-setup-group) | Print settings |
| [Scale to Fit](#scale-to-fit-group) | Fit content on pages |
| [Sheet Options](#sheet-options-group) | Gridlines and headings |
| [Arrange](#arrange-group) | Object positioning |
