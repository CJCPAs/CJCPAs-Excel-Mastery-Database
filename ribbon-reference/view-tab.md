# View Tab - Complete Reference

> **Control how you see your workbook: views, zoom, windows, and macros**

---

## Tab Overview

The View tab controls your visual workspace:

| Group | Purpose |
|-------|---------|
| [Workbook Views](#workbook-views-group) | Different view modes |
| [Show](#show-group) | Toggle UI elements |
| [Zoom](#zoom-group) | Magnification |
| [Window](#window-group) | Multiple windows |
| [Macros](#macros-group) | Run and manage macros |

### Freeze Panes
| Item | Description | Shortcut |
|------|-------------|----------|
| **Freeze Panes** | Lock rows/columns | - |

#### Freeze Options
| Option | Description |
|--------|-------------|
| **Freeze Panes** | Freeze above and left of active cell |
| **Freeze Top Row** | Keep first row visible |
| **Freeze First Column** | Keep first column visible |
| **Unfreeze Panes** | Remove freeze |

#### Freeze Behavior
| Location | What Freezes |
|----------|-------------|
| **Cell B2** | Row 1 and Column A |
| **Cell A5** | Rows 1-4 |
| **Cell C1** | Columns A-B |
| **Cell D4** | Rows 1-3 and Columns A-C |

|------|--------|
