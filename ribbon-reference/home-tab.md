# Home Tab - Complete Reference

> **The most-used tab: formatting, editing, and essential tools**

**Ribbon Access:** Press **Alt+H** to activate Home tab

---

## Tab Overview

The Home tab contains the most frequently used Excel commands, organized into seven groups:

| Group | Purpose | Alt Key |
|-------|---------|---------|
| [Clipboard](#clipboard-group) | Cut, Copy, Paste operations | Alt+H |
| [Font](#font-group) | Text appearance and formatting | Alt+H |
| [Alignment](#alignment-group) | Cell content positioning | Alt+H |
| [Number](#number-group) | Number formatting | Alt+H |
| [Styles](#styles-group) | Quick formatting and tables | Alt+H |
| [Cells](#cells-group) | Insert, delete, format cells | Alt+H |
| [Editing](#editing-group) | Find, sort, fill, clear | Alt+H |

---

## Clipboard Group

| Item | Description | Alt Shortcut | Ctrl Shortcut |
|------|-------------|--------------|---------------|
| **Cut** | Remove selection and place on clipboard | Alt+H, X | Ctrl+X |
| **Copy** | Copy selection to clipboard | Alt+H, C | Ctrl+C |
| **Paste** | Paste clipboard contents | Alt+H, V | Ctrl+V |
| **Paste Values** | Paste only values (no formulas) | Alt+H, V, V | Ctrl+Alt+V, V |
| **Paste Formulas** | Paste only formulas | Alt+H, V, F | Ctrl+Alt+V, F |
| **Paste Formatting** | Paste only formatting | Alt+H, V, T | Ctrl+Alt+V, T |
| **Paste Special...** | Opens Paste Special dialog | Alt+H, V, S | Ctrl+Alt+V |
| **Paste Link** | Paste as linked reference | Alt+H, V, L | - |
| **Paste as Picture** | Paste as image | Alt+H, V, U | - |
| **Format Painter** | Copy formatting to other cells | Alt+H, F, P | - |

| Option | Description | Alt Key in Dialog |
|--------|-------------|-------------------|

| **Validation** | Only data validation rules | N |
| **Column widths** | Only column width settings | W |
