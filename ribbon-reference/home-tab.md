# Home Tab - Complete Reference

> **The most-used tab: formatting, editing, and essential tools**

---

## Tab Overview

The Home tab contains the most frequently used Excel commands, organized into seven groups:

| Group | Purpose |
|-------|---------|
| [Clipboard](#clipboard-group) | Cut, Copy, Paste operations |
| [Font](#font-group) | Text appearance and formatting |
| [Alignment](#alignment-group) | Cell content positioning |
| [Number](#number-group) | Number formatting |
| [Styles](#styles-group) | Quick formatting and tables |
| [Cells](#cells-group) | Insert, delete, format cells |
| [Editing](#editing-group) | Find, sort, fill, clear |

---
