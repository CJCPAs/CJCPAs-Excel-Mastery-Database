## Getting Effective Help

### Before Asking for Help
| Step | Action |
|------|--------|
| **1** | Try F1 or Tell Me first |
| **2** | Search the error message |
| **3** | Check formula syntax |
| **4** | Test with sample data |
| **5** | Note Excel version |

### When Asking for Help
| Include | Description |
|---------|-------------|
| **Excel version** | 365, 2021, 2019, etc. |
| **Operating system** | Windows, Mac |
| **What you tried** | Steps taken |
| **Expected result** | What should happen |
| **Actual result** | What actually happened |
| **Error message** | Exact text |
| **Sample data** | Anonymized example |

## Troubleshooting Common Issues

### Quick Fixes
| Issue | First Try |
|-------|-----------|
| **Formula not calculating** | Check calculation mode (F9) |
| **#REF! error** | Check for deleted cells |
| **#N/A error** | Check lookup value exists |
| **#VALUE! error** | Check data types |
| **Slow performance** | Check volatile functions |
| **File won't open** | Try Open and Repair |

### Getting More Help
1. Check [Troubleshooting Guide](../troubleshooting/README.md)
2. Search Microsoft Support
3. Post in community forums
4. Contact Microsoft Support
