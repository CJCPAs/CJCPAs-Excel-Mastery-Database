# Insert Tab - Complete Reference

> **Add objects, charts, tables, illustrations, and more to your worksheet**

---

## Tab Overview

The Insert tab lets you add various objects to your worksheet:

| Group | Purpose |
|-------|---------|
| [Tables](#tables-group) | PivotTables and Tables |
| [Illustrations](#illustrations-group) | Pictures, Shapes, Icons, 3D Models |
| [Add-ins](#add-ins-group) | Office Add-ins and Store |
| [Charts](#charts-group) | All chart types |
| [Sparklines](#sparklines-group) | Mini in-cell charts |
| [Filters](#filters-group) | Slicers and Timelines |
| [Links](#links-group) | Hyperlinks and Bookmarks |
| [Comments](#comments-group) | Comments and Notes |
| [Text](#text-group) | Text boxes, Headers, WordArt |
| [Symbols](#symbols-group) | Special characters and Equations |
