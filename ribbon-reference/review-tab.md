## Cell Protection Quick Reference

### To Lock Specific Cells Only
1. Select all cells (Ctrl+A)
2. Open Format Cells (Ctrl+1)
3. Go to Protection tab
4. Uncheck "Locked"
5. Click OK
6. Select cells TO lock
7. Format Cells ‚Üí Protection ‚Üí check "Locked"
8. Review ‚Üí Protect Sheet

### To Hide Formulas
1. Select formula cells
2. Format Cells ‚Üí Protection
3. Check "Hidden"
4. Protect the sheet
